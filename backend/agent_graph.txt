  ─────────────────────────────────────────────────────────────────────
  State fields flowing through the graph   (AgentState TypedDict)
  ─────────────────────────────────────────────────────────────────────

   INPUT ──────────────────────────────────────────────────────────────
   raw_diff          str          raw unified diff text from client

   INTERMEDIATE ───────────────────────────────────────────────────────
   parsed_files      list[dict]   set by parse_diff
   reviews           list[...]    appended by each reviewer (operator.add)
   status            str          latest human-readable progress message

   OUTPUT ─────────────────────────────────────────────────────────────
   suggested_patch   str          unified diff patch (generate_patch)
   overall_summary   str          prose summary       (generate_summary)
   test_suggestions  str          test recommendations (generate_summary)


  ─────────────────────────────────────────────────────────────────────
  SSE events emitted to the frontend during /api/review/stream
  ─────────────────────────────────────────────────────────────────────

   event: status   {"node": "<node_name>", "message": "<status text>"}
   event: review   {"category": "...", "issues": [...], "summary": "..."}
   event: patch    {"patch": "<unified diff>"}
   event: summary  {"overall_summary": "...", "test_suggestions": "..."}
   event: done     {}
